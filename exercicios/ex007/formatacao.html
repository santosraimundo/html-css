Principais formas de Formatação de Texto são:

Negrito/Destaque

A linguagem HTML é Semântica por este motivo a frase foi formatada com a tag STRONG (é Semântica).

A CSS é linguagem de estilo e forma esta frase foi formata com a tag B (Não Semântica).

Itálico/Ênfase

Esta frase foi formatada com a tag I (Não Semântica).

Nesta frase formatamos com a tag EM (é Semântica).

Texto Marcado

Nesta frase foi formatada com a tag Mark (é Semântica).

Texto Grande e Pequeno

Nesta frase formatamos o texto com a tag Big que esta descontinuado(Obsoleto) e não é Semântica.

Esta frase foi formatada com a tag Small (é Semântica).

Texto Deletado(Excluido)

Para excluir um texto utilizamos a tag Del (é Semântica).

Texto Inserido/Sublinhado

Esta frase foi sublinhado com a tag U (Não Semântica).

Nesta outra foi sublinhado com a tag Ins (é Semântica).

Texto Sobreescrito

Quando usamos um texto Sobreescrito é para colocar um texto em cima de um outro, utilizamos a tag Sup x2 (é Semântica).

Texto Subescrito

Quando usamos um texto Subescrito colocamos um argumento ou Caracter por do outro Ph (é Semântica).
