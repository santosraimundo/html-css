Comando do MySql.

Classe dos comandos

Comando DDL (Data Definition Language)

Os comandos são: Create database, Create table, Alter table, Drop table ou Drop database.

Comando DML (Data Manipulation Language)

Os comandos de manipulação são: Update, Insert into, Delete, Truncate table

Comando DQL(Data Query Language)

Os comandos solicitação são: Select From, Between, Order by, Like, Max, Min, Count, AVG, SUM, Group by, Having, Inner Join, Left Join, Right Join.
